# Guión Demo Final

## Preparación
1. Abrir navegador en http://localhost:5173
2. Login como admin@example.com

## Demo Flujo Completo

### 1. Login
- Ingresar credenciales
- Mostrar redirección a dashboard

### 2. Dashboard Admin
- Mostrar métricas generales
- Navegar a gestión de agentes

### 3. Gestión de Agentes
- Crear nuevo agente
- Editar información
- Asignar clientes

### 4. Calendario
- Mostrar vistas (día/semana/mes)
- Crear cita con drag & drop
- Mover cita entre horarios

### 5. Notificaciones
- Mostrar banner de recordatorio
- Marcar como leída

## Tiempo Estimado
- Total: 10-15 minutos
